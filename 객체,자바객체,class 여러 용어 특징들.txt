현실 객체 :  미세먼지, 판소리, 사람, 키보드 등등 

객체 지향 언어 : 독립적으로 각각 구분해서 인식할 수 있게끔 하는것을 목표로 하는 것?

자바 객체 : new 연산자를 통해 새로운 객체를 heap 영역에 만든 것 여기서 변수는 heap영역에 있는 객체를 참조하게 되는
참조변수라고도 불리운다.

class : 객체의 특성에 대해 정의를 내린 것

new 연산자 : 객체를 생성하는 데 사용한다 생성된 객체는 heap영역에 만들어진다.

추상화 : 공통으로 가지고 있는 속성값만 맴버변수로 갖는것

캡슐화 : 외부에서 접근하지 못하도록 제한을 걸어두는 것 

캡슐화 원칙 : 맴버변수 앞에 private를 사용한다, 계산값은 class 내부에 사용한다.

메서드/기능의 정의란 : 해당 메서드가 어떤 기능을 수행할 것인지 작성하는것

메서드의 호출이란 :  정의된 메서드의 내용대로 수행 메서드 명을 부르는 것 

return : return 뒤에 적힌값을 가지고 함수를 return 함수 내부에 있는 곳으로 가는 것

public String ex1(){
return "Hi"
};
위 메서드 상세히 해석 :  ex1() 이라는 String 변수를 가지고 있는 함수가있는데 이 함수를 호출하면 hi 라는 문자열을 가지고
호출한 곳으로 간다.
